test1项目使用了JSP和JDBC相结合的经典Web应用开发模式。这种模式的优点是实现相对简单，适合小型项目，但随着需求增大可能面临可维护性和扩展性的问题。     

test2项目JSP、Servlet、JDBC、MySQL数据库和JSTL标签库等技术，结构上遵循了MVC模式，使用Maven管理项目。这样的技术选型适合构建小型Java Web应用，便于实现动态页面展示和数据库交互。
